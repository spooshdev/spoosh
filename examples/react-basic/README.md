# React Basic Example

A minimal Vite React example for Spoosh with:

- `useRead`
- `useWrite`
- `usePages`
- `cache`, `retry`, `invalidation`, and `devtool`
- `msw` for mocked network calls

## Run

1. Install workspace deps from repo root:

```bash
pnpm install
```

2. Start dev server:

```bash
pnpm --filter @spoosh/example-react-basic dev
```
